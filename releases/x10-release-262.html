This is the release page for X10 2.6.2.

The primary features of the X10 2.6.2 release are support for Managed X10 on Java 11 and Native X10 on MacOS 10.12+. Please see the Release Notes for details.

  • X10 Global Matrix Library.  An X10 class library that provides an implementation of sparse and dense distributed matrices.
