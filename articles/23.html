The X10 Test Suite

 

The X10 test suite is available in x10.tests and includes three test scripts in the /bin directory:
  • installverifyScript - confirms proper installation
  • benchmarkScript - executes all benchmark tests and logs timing details in x10.tests/examples
  • fullsuiteScript - executes the complete X10 test suite

 

The X10 test suite encompasses over 1200 test cases and a dedicated test team is adding tests weekly. If you have test cases you wish to share, please contact Jim Xia jimxia@ca.ibm.com

 

Nightly Regression Testing

Automated nightly tests are performed on a variety of platforms. The results are sent via email to the x10-regression mailing list. This is available on SourceForge .
