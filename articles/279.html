X10 Release 2.5.3

This is the release page for X10 2.5.3.

The primary new features of the X10 2.5.3 release are improved compiler support for X10DT features. The X10 Editor is now much more resilient to syntax errors: features such as outline, hyperlink, hover help, and content-assist continue to work in the face of compilation errors. Please see the Release Notes for details.
