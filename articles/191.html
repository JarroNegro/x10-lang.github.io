A debugger for X10 versions 2.2.x is  available for download from IBM developerWorks. Note that the debugger does not support X10 version 2.3.0 or later. The debugger integrates with X10DT and supports debugging X10 programs compiled with the C++ backend and executing on Linux/Power, Linux/x86, and Linux/x86_64.  The debugger GUI is supported on all platforms that are supported by X10DT.  The debugger supports remote and multi-node debugging of the executing X10 program. 

For more information about the debugger including a movie that demonstrates its features, please see the IBM Parallel Debugger for X10 Programming project . at IBM developerWorks.
