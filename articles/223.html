--- layout: page title: "Open Source release of ScaleGraph 1.0 Released" alias: "open-source-release-of-scalegraph-1-0-released" category: "news/latest-news" date: 2012-10-14 20:05:00 ---

The ScaleGraph project has made an initial open source release of ScaleGraph version 1.0.  ScaleGraph is an X10 library that  provides support for both managing and analyzing large graphs in a scalable fashion. More details are available at ScaleGraph.org and in an X10'12 workshop paper by the ScaleGraph team.
