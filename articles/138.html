Version 2.0.5.1 of X10 is now available for download.

The 2.0.5.1 release corrects the issues below, but is otherwise unchanged from the 2.0.5 release.

  • Only one level of stacktraces was printed on MacOS
  • With the C++ backend, if application code contained X10 assert statements and was compiled without optimization, it would not link properly against the pre-built release binaries.
  • A bug in closure inlining potentially resulting in duplicate labels in the C++ code generated by the C++ backend.
  • Compilation problems with the math.h header file on FreeBSD

Included at the top-level of the X10 release is a directory of sample programs.

  • Source Distribution of X10 2.0.5.1 Use this if you want to build X10 yourself, modify the X10 implementation, or need X10 for a platform that is not one of the ones for which we provide pre-built binaries.
  • X10DT 2.0.5 - see the Installation and basic setup instructions here.

Instructions

  • Download the X10 2.0.5.1 distribution for your platform.
  • Untar it.
  • See the INSTALL file in the X10 2.0.5.1 directory tree for additional instructions.

Binary builds of the Common PGAS runtime that match the X10 2.0.5.1 release are available as separate binary downloads. Under most circumstances, you should not need to manually download the Common PGAS runtime yourself. It is included in the binary X10 releases and will be automatically downloaded by the X10 build process if you are building from source.

Erratum

This can be resolved by running as root:

echo 127.0.0.1 `hostname` &gt;&gt; /etc/hosts
