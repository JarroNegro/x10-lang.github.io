As announced on the x10-users list, ScaleGraph 2.1 has been released.  ScaleGraph is a graph library on top of the state-of-the-art X10 programming language for large-scale graph processing. Features at a glance:

  • XPregel ­– a framework for Pregel computational model
  • Degree Distribution
  • Page Rank
  • Vertex Betweenness Centrality
  • Spectral Clustering

For more information, please refer to the official web site at scalegraph.org.
