APGAS Release 1.0.0

This is the release page for APGAS 1.0.0.

The APGAS library supports the APGAS programming model in Java 8. See http://x10.sourceforge.net/documentation/papers/X10Workshop2015/tardieu.pdf for details.

The simplest way to start out with APGAS is by installing the APGAS Development Tools for Eclipse 4.4.2 using the APGAS update site or by downloading an archive of the update site.
