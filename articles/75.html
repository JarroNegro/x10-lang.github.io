Xiaoge Wang, Tsinghua University, Beijing, China (X10 Team Contact: Haibo Lin).

&nbsp;This project is to develop courseware for a short course on parallel programming with X10. The courseware could be used for a stand alone short course on parallel programming with X10, or be integrated with other materials, such as MPI and OpenMP, to comprise the courseware for a typical one-semester parallel programming course. It would contain lecture notes, program examples, hands-on-exercises, problem sets with sample solutions and a set of term-project topics including the reference report of the term-project.
